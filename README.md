# EssentialApps

This project is structured as a monorepo with the following directories:

- `client/`: The React Native / Expo frontend application.
- `server/`: The backend server (implementation pending).

## Getting Started

### Client

Navigate to the `client` directory to run the frontend:

```bash
cd client
npm install
npm start
```

### Server

Navigate to the `server` directory to run the backend:
